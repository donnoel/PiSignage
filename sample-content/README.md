sample-content is the tracked seed content for fresh clones and new Pi installs.

The tracked seed playlist contains a single Beam fallback loop. A new Pi should
show this local playback-safe MP4 on first boot, before a screen has been assigned
or a customer playlist has been published from Beam.

Media in sample-content/assets is local uploaded/demo media and remains ignored by
git, except for the tracked `beam-ready.signage-1080p.mp4` first-run fallback.
Do not add customer or production media here.
